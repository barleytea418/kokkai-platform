# 起動方法

## 1. Node.js のインストール

以下のいずれかの方法でインストールしてください。

### 方法A: 公式インストーラー（推奨・簡単）
https://nodejs.org/ja/ にアクセスし「LTS版」をダウンロード・インストール

### 方法B: Homebrew を使う場合
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node
```

## 2. プロジェクトの起動

ターミナルで以下を実行：

```bash
cd ~/Documents/kokkai-platform
npm install
npm run dev
```

## 3. ブラウザで確認

http://localhost:3000 を開く
